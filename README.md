# Translate Mania

A linguistic survival game where you must translate phrases to survive. Practice vocabulary, learn grammar, and test your translation skills across multiple languages.

## Features

### Core Gameplay
- **Multiple Game Modes**: 
  - Classic Mode - 20 phrases to translate
  - Daily Challenge - New challenge every day (15 phrases)
  - Practice Mode - Learn at your own pace (40 phrases, no timer)
  - Maniac Mode - Random FROM/TO language pairs (25 phrases)

- **5 Languages Supported**: Spanish, Greek, English, German, Chinese
- **3 Lives System**: Start with 3 lives, lose one for each incorrect answer

### Difficulty Levels
- Easy, Medium, Hard, Expert - phrases scale with your choice

### Power-ups & Shop
- Hint, Time Freeze, Skip, Shield
- Double Points, Extra Life, Timer +10s
- Health Potion, XP Boost, and more!
- Purchase with XP earned from correct answers

### Learning Features
- **Text-to-Speech**: Listen to phrases in both FROM and TO languages
- **Pinyin Support**: Chinese characters display with pinyin romanization
- **Vocabulary & Grammar**: Built into Stats screen with external resource links
- **Error History**: Track your mistakes and learn from them
- **Progress Tracking**: XP, levels, streaks, accuracy stats

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Canvas Confetti

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Game Controls

- Click option or press 1-4 to select answer
- Use power-ups (Hint, Freeze, Skip, Shield) during gameplay
- Click speaker icon to hear pronunciation
