# Suggested Updates & Feature Ideas

This document outlines features that could dramatically enhance Polyglot Peril, inspired by Duolingo but made more difficult and intense.

---

## 1. Game Modes

### 1.1 Daily Challenge (Like Duolingo's Leagues)
- **Concept**: 10-15 phrases per day shared globally
- **Difficulty**: Mix of all 3 difficulties (5 Easy, 5 Medium, 5 Hard)
- **Leaderboard**: Global rankings updated daily
- **Streak Sharing**: Share your daily streak on social media
- **Time Window**: 24-hour window to complete

### 1.2 Speed Run
- **Concept**: Answer as many as possible in 60 seconds
- **Endless Mode**: Phrases keep generating until timeout
- **Leaderboard**: Track highest phrases completed
- **Bonus**: +1 second per 3 correct answers (like Duolingo's timed quizzes)

### 1.3 Endurance Mode (Iron Mode)
- **Concept**: No lives system - continue until 3 wrong in a row
- **Progressive Difficulty**: Gets harder after every 20 correct
- **Ranking**: Track highest level reached
- **Badge Unlocked**: "Iron Will" achievement at 100+ streaks

### 1.4 Practice Mode (Casual)
- **Concept**: No timer, no lives, no score pressure
- **Learning Focus**: Show correct answer after each attempt
- **Grammar Notes**: Pop-up explanations after each phrase
- **Weakness Tracking**: Highlight phrases you struggle with
- **Review Later**: Flag phrases for flashcard review

### 1.5 Story Mode (Narrative-Driven)
- **Concept**: Sequential story chapters unlock over time
- **Progressive Narrative**: Dark owl abduction story unfolds
- **Multiple Endings**: Choices affect story outcome
- ** Dialogue Translation**: Translate dialogue to progress story

### 1.6 Exam Prep Mode
- **Concept**: Prepare for specific proficiency tests (DELE, DELF, JLPT, HSK)
- **Test-Like Format**: Mimics actual exam structure
- **Timed Sections**: Practice under exam conditions
- **Score Estimates**: Show approximate proficiency level

## 3. UI/UX Improvements

### 3.1 Visual Redesign (Duolingo-Inspired)

#### 3.1.1 Color Scheme
- **Primary**: Green (#58CC02) for success/correct
- **Secondary**: Red (#FF4B4B) for errors/wrong
- **Accent**: Gold (#FFC800) for streaks/rewards
- **Background**: Light gray (#F7F7F7) or dark (#1C1C1C) for dark mode

#### 3.1.2 Character/Mascot
- Keep the owl mascot (can be styled differently per theme)
- Add animations for correct answers (bounce, celebrate)
- Add sad animations for wrong answers
- **Owl Expressions**: Happy, disappointed, encouraging, proud

#### 3.1.3 Interface Elements
- **Progress Bars**: Rounded, cartoonish style
- **Hearts/Lives**: Displayed as hearts (like Duolingo)
- **XP Counter**: Visible at all times with animations
- **Streak Counter**: Flame icon with fire animation
- **Level Indicator**: Prominent player level display

### 3.2 Feedback Improvements

#### 3.2.1 Answer Feedback
- **Correct**: Green glow, checkmark animation, sound effect
- **Wrong**: Red shake, X animation, correct answer reveal
- **Perfect**: Golden sparkle effect for 5+ streak

#### 3.2.2 Progress Animations
- **XP Pop-ups**: +XP floats up on correct answers
- **Level Up**: Celebration screen when leveling up
- **Streak Milestone**: Fire animation at milestones
- **Achievement Unlock**: Badge unlock animations

### 3.3 Accessibility

- **High Contrast Mode**: Toggle for accessibility
- **Larger Text**: Adjustable font sizes
- **Screen Reader**: ARIA labels and support
- **Color Blind Friendly**: Alternative color schemes
- **Keyboard Navigation**: Full keyboard support
- **Reduced Motion**: Disable animations option

### 3.4 Dark Mode
- **System Follow**: Auto-detect system preference
- **Manual Toggle**: Switch between light/dark
- **OLED Black**: True black for AMOLED screens

### 3.5 Sound Design

- **Correct Sound**: Pleasant chime/ding
- **Wrong Sound**: Gentle buzz (not harsh)
- **Level Up**: Celebratory jingle
- **Timer Warning**: Tick-tock when time low
- **Streak Fire**: Crackling fire sound
- **Sound Toggle**: Mute button prominent
- **Volume Control**: Adjustable sound levels
- **Background Music**: Optional ambient music

## 6. Progression System

### 6.1 Player Levels
- **XP Thresholds**: Level up at 1000, 3000, 6000, 10000 XP...
- **Level Rewards**: Unlock new features at each level
- **Tiered Titles**:
  - Level 1-9: Novice
  - Level 10-19: Apprentice
  - Level 20-29: Intermediate
  - Level 30-39: Advanced
  - Level 40-49: Expert
  - Level 50+: Master

### 6.2 Skill Trees (Unlockable Paths)
- **Linguist Tree**: Focus on vocabulary
- **Grammar Tree**: Focus on grammar rules
- **Speed Tree**: Focus on fast answers
- **Endurance Tree**: Focus on streaks

### 6.3 Journey/Unlocking
- **Languages**: Unlock new languages at certain levels
- **Features**: Unlock new modes as you progress
- **Content**: New phrase banks unlock over time

- **Speed Penalty**: Lose streak bonus for slow answers
- **Combo System**: Must maintain 3+ streak for bonus points

### 7.2 Mastery Checks

- Combine multiple concepts in single question
- Reverse translation (target language to English)
- Fill in the blank

- Weekly competitions
